# Design Context — ConsultantPlatform (MindMuse)

## Audience
Psychosocial consultants processing sensitive consultation material (transcripts, themes, evidence). Desktop-first, professional work sessions.

## Use cases
Primary home is conversational: upload material, review AI outputs, confirm records without leaving chat. Sidebar remains for deep navigation when needed.

## Brand personality
Clinical neutrality, calm authority, generous whitespace. No celebration language, urgency, or generic “AI assistant” chrome. Feels like a focused workspace, not a chatbot demo.

## Visual system
- Shadcn preset `aKpFZLe` — use existing CSS variables in `app/globals.css` (OKLCH tokens)
- Typography: existing app sans stack; fixed rem scale for UI
- Layout: minimal borders, strong hierarchy, stats secondary to primary actions
- Motion: subtle only (thinking dots, collapsible stats)

## Differentiation
Action-first welcome state; stats collapse behind a quiet control. Thread + fixed input bar as the single focal column.
